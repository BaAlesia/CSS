Простые селекторы: практика
1. Селекторы по тегу
Пример: задай всем элементам <a> красный цвет текста. Решение:

a {
  color: red;
}
Сделай самостоятельно: задай всем элементам <p> красный цвет текста.

Сделай самостоятельно: задай всем элементам <a> и <p> красный цвет текста.

 Решение:
 p {
  color: red;
}
a, p {
  color: red;
}

2. Селекторы по классу
Пример: задай всем элементам с class="foo" красный цвет текста. Решение:

.foo {
  color: red;
}
Сделай самостоятельно: задай всем элементам bar красный цвет текста и подчеркни текст (гугли).

 Решение:
 .bar {
  color: red; text-decoration: underline;
}

3. Селекторы по id
Пример: задай всем элементам с id="foo" красный цвет текста. Решение:

#foo {
  color: red;
}
Сделай самостоятельно: задай элементу с id="green" зелёный цвет фона.

 Решение:
 #green {
  background: green;
}
